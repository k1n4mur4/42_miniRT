# libft

`libft`は、プログラミングスクール42のカリキュラムの一環として作成された、C言語のカスタムライブラリです。このプロジェクトは、多くの標準Cライブラリ関数を再実装し、さらに便利なユーティリティ関数を追加することを目的としています。

このライブラリには、`ft_printf`と`get_next_line`の実装も含まれています。

## 機能一覧

このライブラリは、機能に基づいていくつかのモジュールに分かれています。

### Ctype関数 (`<ctype.h>`)

- `ft_isalnum`: 英数字かどうかをチェックします。
- `ft_isalpha`: 英字かどうかをチェックします。
- `ft_isascii`: 7ビットUS-ASCII文字かどうかをチェックします。
- `ft_isdigit`: 数字 (0-9) かどうかをチェックします。
- `ft_ishexdigit`: 16進数の文字かどうかをチェックします。
- `ft_isprint`: 表示可能な文字かどうかをチェックします。
- `ft_isspace`: 空白文字かどうかをチェックします。
- `ft_toupper`: 小文字を対応する大文字に変換します。
- `ft_tolower`: 大文字を対応する小文字に変換します。

### 文字列操作関数 (`<string.h>`)

- `ft_split`: 指定された区切り文字で文字列を分割します。
- `ft_strchr`: 文字列内で最初に出現する文字を検索します。
- `ft_strcmp`: 2つの文字列を比較します。
- `ft_strdup`: 文字列を複製します。
- `ft_striteri`: 文字列の各文字に特定の関数を適用します。
- `ft_strjoin`: 2つの文字列を連結します。
- `ft_strlcat`: サイズをチェックしながら、文字列をバッファに追加します。
- `ft_strlcpy`: サイズをチェックしながら、文字列をバッファにコピーします。
- `ft_strlen`: 文字列の長さを計算します。
- `ft_strmapi`: 文字列の各文字に関数を適用し、新しい文字列を生成します。
- `ft_strncmp`: 2つの文字列をn文字まで比較します。
- `ft_strndup`: 文字列をn文字まで複製します。
- `ft_strnstr`: 文字列内から部分文字列を検索します。
- `ft_strrchr`: 文字列内で最後に出現する文字を検索します。
- `ft_strtrim`: 文字列の先頭と末尾の空白文字を削除します。
- `ft_substr`: 文字列から部分文字列を抽出します。

### メモリ操作関数 (`<string.h>` / `<strings.h>`)

- `ft_bzero`: バイト文字列をゼロで埋めます。
- `ft_memchr`: メモリ領域から文字を検索します。
- `ft_memcmp`: 2つのメモリ領域を比較します。
- `ft_memcpy`: メモリ領域をコピーします。
- `ft_memmove`: 重なり合う領域にも対応してメモリ領域をコピーします。
- `ft_memset`: メモリを特定のバイトで埋めます。

### 標準ライブラリ関数 (`<stdlib.h>`)

- `ft_atoi`: 文字列を整数 (int) に変換します。
- `ft_atol`: 文字列を長整数 (long) に変換します。
- `ft_calloc`: 配列のメモリを確保し、ゼロで初期化します。
- `ft_hextoi`: 16進数の文字を整数に変換します。
- `ft_itoa`: 整数を文字列に変換します。
- `ft_strtol`: 文字列を基数を指定して長整数 (long) に変換します。

### I/O関数

- `ft_putchar_fd`: 指定されたファイルディスクリプタに文字を出力します。
- `ft_putstr_fd`: 指定されたファイルディスクリプタに文字列を出力します。
- `ft_putendl_fd`: 指定されたファイルディスクリプタに文字列と改行を出力します。
- `ft_putnbr_fd`: 指定されたファイルディスクリプタに整数を出力します。

### 連結リスト関数

- `ft_lstnew`: 新しいリスト要素を作成します。
- `ft_lstadd_front`: リストの先頭に要素を追加します。
- `ft_lstadd_back`: リストの末尾に要素を追加します。
- `ft_lstsize`: リストの要素数を数えます。
- `ft_lstlast`: リストの最後の要素を返します。
- `ft_lstdelone`: リストの単一の要素を削除します。
- `ft_lstclear`: リスト全体を削除して解放します。
- `ft_lstiter`: リストの各要素に関数を適用します。
- `ft_lstmap`: 既存のリストの各要素に関数を適用し、新しいリストを生成します。

### 追加プロジェクト

- **`get_next_line`**: ファイルディスクリプタから一行ずつ読み込みます。
- **`ft_printf`**: `printf`関数を再実装し、様々なフォーマット指定子に対応します。

## ビルド方法

ライブラリをビルドするには、プロジェクトのルートディレクトリで`make`を実行します。

```sh
make
```

これにより、ソースファイルがコンパイルされ、`libft.a`という名前のスタティックライブラリが作成されます。

### その他の `make` コマンド

- `make clean`: オブジェクトファイル (`.o`) を削除します。
- `make fclean`: オブジェクトファイルと`libft.a`ライブラリを削除します。
- `make re`: ライブラリをクリーンな状態から再ビルドします (`fclean` + `all`)。

## 使い方

1.  作成した`libft.a`ライブラリと一緒に、あなたのプロジェクトファイルをコンパイルします。
2.  ソースファイル内で`libft.h`ヘッダーをインクルードします。

### 使用例

以下は、このライブラリを使用した`main.c`の例です。

```c
#include "includes/libft.h"
#include <stdio.h>

int main(void)
{
    char *str = "   こんにちは, 世界!   ";
    char *trimmed_str;

    trimmed_str = ft_strtrim(str, " ");
    if (trimmed_str)
    {
        printf("元の文字列: '%s'\n", str);
        printf("トリム後の文字列: '%s'\n", trimmed_str);
        free(trimmed_str);
    }
    return (0);
}
```

このサンプルをコンパイルするには、以下のコマンドを使用します。

```sh
# まず、makeを実行してlibft.aをビルドします
make

# 次に、mainプログラムをコンパイルし、ライブラリとリンクします
gcc main.c -L. -lft -o my_program
```

そして、プログラムを実行します。

```sh
./my_program
```
